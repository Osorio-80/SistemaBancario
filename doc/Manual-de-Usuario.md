# Manual de Usuario

## Registrar una cuenta

1. Seleccionar el tipo en el ComboBox (Ahorros / Corriente / CDT)
2. Ingresar número de cuenta (solo números)
3. Ingresar nombre del titular (solo letras)
4. Ingresar saldo inicial
5. Si es CDT: ingresar plazo en meses y meses transcurridos
6. Presionar "＋ Registrar Cuenta"

## Depositar

1. Ingresar el número de cuenta en la sección Operaciones
2. Ingresar el monto a depositar
3. Presionar "↑ Depositar"
   - CDT: mostrará mensaje de bloqueo (RN-02)

## Retirar

1. Ingresar número de cuenta y monto
2. Presionar "↓ Retirar"
   - Ahorros: retira si hay saldo suficiente
   - Corriente: descuenta monto + 4x1000
   - CDT: bloquea si no cumplió el plazo

## Ver resumen

- Ver Todas → muestra todas las cuentas con interés
- Mi Cuenta → muestra la cuenta específica con interés

## Indicadores de color

- Verde  → operación exitosa
- Rojo   → error o saldo insuficiente
- Ámbar  → advertencia (CDT bloqueado)
- Gris   → información neutral
